01Overview

Adopt STRATA (source: strata-design-system/tokens.ts, design-system.md, StudyTracker.jsx) as LevelUp's design system, replacing the current emerald/amber light+dark Tailwind theme entirely. STRATA is a dark-only "field research console" aesthetic: near-black instrument panel, status dots instead of checkboxes, IBM Plex Mono for all data/numbers, Fraunces used sparingly for section titles only, and a stamp-in micro-animation on checklist completion.

Decisions locked during brainstorming — see list below. Each was confirmed by the user before this spec was written.
  • Dark-only. No light mode, no dark: variant anywhere.
  • STRATA's color semantics fully adopted — amber = active/in-progress/primary; emerald = complete-only.
  • Visual only — existing LevelUp terminology (Topics, Lessons, Rewards, Badges) is unchanged.
  • Scope: foundation + full restyle in one pass, ~33 components.
  • Architecture: Tailwind v4 @theme tokens drive all utilities, minimal global.css.
  • Fonts: static <link> in Base.astro head, no runtime JS.
  • PWA icons regenerated in this pass via existing script.
  • ThemeToggle.svelte removed; theme field stays inert in AppData.

global.css changes

  • Remove @custom-variant dark entirely; drop every dark: prefix across all components.
  • body → bg-ink text-text-primary font-body antialiased, no conditional branch.
  • Stamp-in keyframe ported from STRATA, plus a prefers-reduced-motion override disabling it and all row/patch hover transitions.
  • Global :focus-visible ring always on: outline: 2px solid var(--color-active).
  • Lesson .prose override: headings use font-display, body stays font-body, code/pre use font-data.
  • Shiki switches from dual-theme to single dark theme: github-dark-default.
  • Markdown callouts remap: tip→amber, note→teal, important→teal (bold), warning→amber (bold), caution→rose.
  • Mermaid themeVariables overridden to use panel/line/text-primary instead of default light theme.
04Component mapping

Shared primitives

  • Status dot — open circle (line) = not-started, filled amber = in-progress, emerald check = complete.
  • Card — bg-panel border border-line rounded-md, no shadows.
  • Row — bg-transparent hover:bg-panel-soft, real <button>.
  • Stat block — mono muted uppercase label + mono semantic-colored value.
  • Patch — locked: 50% opacity, panel-soft/line. Unlocked: full opacity, amber-soft/amber.
  • Progress bar — track line, fill amber, flips to emerald only at 100%.
